# SelfBuildUp-SE
당신이 막 취직한 소프트웨어 엔지니어이거나, 또는 학부생이라면 컴퓨터과학을 공부하면서 한번쯤은 막히는 경험이 있을 겁니다.
다행스럽게도, 컴퓨터과학의 지식들은 대부분 온라인 상에서 얻기가 매우 쉽다는 것입니다. 전공을 배우다가 막혔거나, 또는 잊었을때 유용하게 다시 배울 수 있습니다.
당신이 필요한 지식을 무료 리소스를 통해서 학습해보세요. 모든 지식들은 무료로 제공됩니다. 

이 자료는 한국인들을 위해 대부분 한글로 이루어진 자료들로 이루어져 있습니다.
[TeachYourselfCS-KR](https://github.com/minnsane/TeachYourselfCS-KR)을 보고 영감을 받아 제작되었습니다.
