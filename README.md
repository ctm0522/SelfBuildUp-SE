## 컴퓨터공학기초
|과목|설명|책|영상|
|---|--------------|-|---|
|이산수학|"컴퓨터 과학의 언어이자 논리적 사고의 뼈대." 연속적인 미적분학이 아닌, 0과 1의 세계를 다루는 이산적인 수학을 다룹니다. 논리, 증명, 그래프 이론은 코드가 '왜' 작동하는지, 그리고 '얼마나' 효율적인지 증명하는 강력한 도구가 됩니다.|-|[http://www.kocw.net/home/search/kemView.do?kemId=1165096](http://www.kocw.net/home/search/kemView.do?kemId=1165096)|
|자료구조|"데이터를 다루는 그릇, 상황에 맞는 도구 선택하기." 단순히 코딩 테스트를 위해서가 아닙니다. 메모리를 효율적으로 쓰고 데이터를 빠르게 검색하기 위해 어떤 구조를 써야 할까요? 스택, 트리, 그래프 등 각 자료구조의 특성과 Trade-off(장단점)를 이해하는 것은 백엔드 성능 최적화의 첫걸음입니다.|-|[https://inf.run/Xe8WS](https://inf.run/Xe8WS)|
|알고리즘|"문제를 해결하는 컴퓨팅 사고력(Computational Thinking)." 자료구조가 '재료'라면 알고리즘은 '요리법'입니다. 정렬, 탐색, 동적 계획법 등을 통해 문제를 추상화하고, 한정된 자원(시간과 메모리) 안에서 가장 빠르고 정확하게 답을 찾아내는 엔지니어의 핵심 역량을 기릅니다.|-|[https://inf.run/nQ9Be](https://inf.run/nQ9Be)|
